#### Пример: Растеризация отрезка (0, 0) — (4, 2)
Параметры: $x_1=0, y_1=0, x_2=4, y_2=2$.
$dx = 4, dy = 2$.

**1. Пошаговый алгоритм**
Уравнение прямой: $y = kx + b$.
$k = dy/dx = 2/4 = 0.5$.
$b = y_1 - kx_1 = 0 - 0 = 0$.
Так как $|dx| > |dy|$, цикл идет по $x$ от 0 до 4.

| Шаг ($x$) | Вычисление $y = 0.5 \cdot x$ | Округление $y_{int} = \text{round}(y)$ | Пиксель |
| :--- | :--- | :--- | :--- |
| $x=0$ | $y = 0$ | 0 | **(0, 0)** |
| $x=1$ | $y = 0.5$ | 1 (обычное округление) | **(1, 1)** |
| $x=2$ | $y = 1.0$ | 1 | **(2, 1)** |
| $x=3$ | $y = 1.5$ | 2 | **(3, 2)** |
| $x=4$ | $y = 2.0$ | 2 | **(4, 2)** |

---

**2. Алгоритм ЦДА (DDA)**
Длина отрезка $L = \max(|dx|, |dy|) = \max(4, 2) = 4$.
Приращения:
$sx = dx/L = 4/4 = 1$.
$sy = dy/L = 2/4 = 0.5$.
Начальные координаты: $x=0, y=0$.

| Шаг $i$ (0..L) | Текущие $x, y$ | Округление (Pixel) | Приращение |
| :--- | :--- | :--- | :--- |
| 0 | $x=0, y=0$ | **(0, 0)** | $x+=1, y+=0.5$ |
| 1 | $x=1, y=0.5$ | **(1, 1)** | $x+=1, y+=0.5$ |
| 2 | $x=2, y=1.0$ | **(2, 1)** | $x+=1, y+=0.5$ |
| 3 | $x=3, y=1.5$ | **(3, 2)** | $x+=1, y+=0.5$ |
| 4 | $x=4, y=2.0$ | **(4, 2)** | - |

---

**3. Алгоритм Брезенхема (Отрезок)**
Входные данные: $(0,0) \to (4,2)$.
$dx = 4, dy = 2$.
Начальная ошибка: $E = 2\cdot dy - dx = 2\cdot 2 - 4 = 0$.
Правила обновления (так как $dx > dy$, идем по $x$):
*   Если $E \ge 0$: $y$ увеличиваем, $E \leftarrow E + 2\cdot(dy - dx)$.
*   Если $E < 0$: $y$ не меняем, $E \leftarrow E + 2\cdot dy$.
*   $x$ увеличиваем на каждом шаге.

$Const_1 = 2(dy - dx) = 2(2-4) = -4$.
$Const_2 = 2dy = 4$.

| Шаг | Текущий Пиксель | Проверка $E$ | Изменение $y$ | Пересчет $E$ |
| :--- | :--- | :--- | :--- | :--- |
| 0 | **(0, 0)** | $E=0 (\ge 0)$ | $y \leftarrow 1$ | $E = 0 - 4 = -4$ |
| 1 | **(1, 1)** | $E=-4 (< 0)$ | $y$ без изм. | $E = -4 + 4 = 0$ |
| 2 | **(2, 1)** | $E=0 (\ge 0)$ | $y \leftarrow 2$ | $E = 0 - 4 = -4$ |
| 3 | **(3, 2)** | $E=-4 (< 0)$ | $y$ без изм. | $E = -4 + 4 = 0$ |
| 4 | **(4, 2)** | (Конец) | | |

#### Пример 2: Алгоритм Брезенхема для окружности (R=3)
Центр $(0,0)$, Радиус $R=3$.
Рисуем дугу 2-го октанта (от $(0, R)$).
Начальные условия: $x=0, y=3$.
Начальная ошибка: $E = 3 - 2R = 3 - 6 = -3$.

Цикл `while (x < y)`:
1.  Если $E \ge 0$: $y \leftarrow y-1$, $E \leftarrow E + 4(x-y) + 10$.
2.  Иначе: $y$ без изм., $E \leftarrow E + 4x + 6$.
3.  $x \leftarrow x+1$.

| Итерация | Текущий пиксель | Анализ ошибки | Обновление $E$ | Новые $x, y$ |
| :--- | :--- | :--- | :--- | :--- |
| Старт | **(0, 3)** | $E = -3 (< 0)$ | $E = -3 + 4(0) + 6 = 3$ | $x=1, y=3$ |
| 1 | **(1, 3)** | $E = 3 (\ge 0)$ | $E = 3 + 4(1-3) + 10 = 3 - 8 + 10 = 5$ | $x=2, y=2$ |
| 2 | **(2, 2)** | Условие $x < y$ (2 < 2) ложно | Цикл завершен | |

**Результат:** Пиксели дуги 2-го октанта: $(0,3), (1,3), (2,2)$. Остальные строятся симметрично.
